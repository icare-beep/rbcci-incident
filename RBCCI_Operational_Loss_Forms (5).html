📖 Quick Setup Guide
  1. Go to Google Cloud Console → create/select a project
  2. Enable Google Sheets API and Google Drive API
  3. Create an OAuth 2.0 Client ID (Web Application type) and an API Key
  4. Add your app URL to Authorized JavaScript origins (e.g. your Railway domain)
  5. Create a Google Spreadsheet with tabs: Loss Event Log, Quarterly Summary, KRI Dashboard, Category Mapping, Incident Report Form
  6. Create a Google Drive folder named Document Vault and copy its folder ID
  7. Paste all IDs above → click Save & Connect → Sign in with Google
