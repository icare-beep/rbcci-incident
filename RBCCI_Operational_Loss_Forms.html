📖 Quick Setup Guide
  1. Go to Google Cloud Console → Create or select a project
  2. Enable APIs: Search and enable Google Sheets API and Google Drive API
  3. Create credentials: APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → Web Application
  4. Add Authorized Origins: Add ${window.location.origin} to Authorized JavaScript origins
  5. Create a Google Spreadsheet with these exact sheet tabs: Loss Event Log, Quarterly Summary, KRI Dashboard, Category Mapping, Incident Report Form
  6. Create a Google Drive folder named Document Vault and copy its folder ID
  7. Paste the Client ID, Spreadsheet ID, and Folder ID above and click Connect

📘 Quick Setup Guide
  1. Go to console.cloud.google.com and create or select a project.
  2. Enable Google Sheets API and Google Drive API under APIs & Services → Library.
  3. Go to APIs & Services → Credentials and create an OAuth 2.0 Client ID (Web application type). Add your site URL as an authorized origin.
  4. Also create an API Key and restrict it to Sheets + Drive APIs.
  5. Create a Google Sheet named "Incident Loss Report" and copy its Spreadsheet ID from the URL.
  6. Create a folder in Google Drive named "Document Vault" and copy its Folder ID from the URL.
  7. Paste the Client ID, API Key, Spreadsheet ID, and Folder ID above, then click Save & Connect.
  8. Click Sign in with Google when it appears and grant Sheets + Drive permissions.
